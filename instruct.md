# Инструкция по GIT

* **git init**

* **git status**

* **git add**

* **git commit -m "Message"**

* **git log**

* **git checkout**

* **git checkout master**

* **git diff**

* **git branch**

* **git merge**

*Команда используется для слияния веток, что по итогу образует целостную ветку без ветвления*
