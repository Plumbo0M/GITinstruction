# Инструкция по GIT

* **git init**

* **git status**

* **git add**

* **git commit -m "Message"**

* **git log**

* **git checkout**

* **git checkout master**

* **git diff**

* **git branch**

1. *Команда используется для создания и управления всеми ветками*

   * Если необходимо удалить ветку используем флаг -d.

   * Для создания ветки используем branch branch_name.   

* **git merge**
